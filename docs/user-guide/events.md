# Events

A date-ordered list of things to go to or remember by calendar date — concerts, appointments, parties, and deadlines tied to a specific day.

## Adding an event

1. Go to **Events** (`/events`)
2. Create a new event with:

| Field | Required | Notes |
| ----- | -------- | ----- |
| Title | Yes | Event name |
| Date/time | Yes | When it happens |
| Location | No | Where |
| Link | No | URL — tickets, calendar link, etc. |
| Note | No | Extra context |

**Examples:**

- "concert at the park — June 21"
- "vet appointment — next Tuesday 3pm"
- "block party"

## Viewing events

Events sort chronologically. The list shows **upcoming** events by default; past events can be viewed on the full page (they archive rather than clutter the present).

## Home page

The home events section shows items **within the next two weeks**.

## vs. the stream

| Use events when… | Use the stream when… |
| ---------------- | -------------------- |
| Something has a specific date/time | Timing is vague ("sometime this week") |
| You want chronological calendar order | It's a general reminder or thought |
