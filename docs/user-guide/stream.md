# Stream

The stream is hearth's catch-all layer — tasks, reminders, and half-formed thoughts anyone can add.

If something doesn't fit restaurants, projects, metrics, events, or inventory yet, it lives here.

## Quick capture

The stream leads with your existing notes; a compact capture box sits above the list so adding is quick without dominating the page.

1. Go to **Stream** (`/stream`)
2. Type your note in the capture box
3. Click **Add**

That's it. No required categories, due dates, or assignments.

**Examples:**

- "pick up more dog food"
- "call about the water heater"
- "remember to bring the crock pot to Sarah's"

## Viewing entries

Entries appear in chronological order (newest first by default). Each entry shows:

- The note body
- Who added it and when
- Pin and done status

Open entries also appear on the **home page** when pinned or recently added.

## Pinning

Pin an entry to surface it on the home page for household attention.

1. Find the entry in the stream list
2. Use the pin action (hover or row menu)
3. Pinned items appear in the home stream section

Unpin when it no longer needs top-of-mind visibility.

## Marking done

When something is handled, mark it done:

1. Use the done action on the entry
2. Done items are visually muted; they remain in history

Done is a soft state — not a delete. You can reopen if needed.

## Editing

Edit the body, rough timing, or metadata on any entry you have access to (all household members can edit shared content).

## Rough when

Optionally add freeform timing context:

- "this week"
- "before the trip"
- "sometime in March"

This is display-only — hearth does not send reminders or alerts based on rough when.

## @-mentions

Type `@username` in a stream note to notify a household member. They'll see it in their notification feed.

See [Notifications & @-mentions](notifications-and-mentions.md) for details.

## Photos

Attach photos to stream entries — receipts, screenshots, or anything visual. See [Attachments](attachments.md).

## Home page summary

The home stream section shows:

- Pinned open items
- Recent additions

Click **View all** to go to the full stream list.
